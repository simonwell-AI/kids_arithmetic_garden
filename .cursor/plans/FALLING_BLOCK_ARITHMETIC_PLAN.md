# 落下方塊算術遊戲 — 實作計畫

依先前討論：方塊從上落下、題目寫在方塊上（如 12+17=?），答對得分、答錯或落地未答則方塊消失換下一塊。

---

## 1. 玩法與規則（第一版）

| 項目 | 說明 |
|------|------|
| **關卡形式** | 單一欄位：一次只落一個方塊，底部一個輸入框。 |
| **方塊內容** | 題目字串，例如 `12+17=?`、`8−3=?`、`4×5=?`、`20÷4=?`。 |
| **答對** | 方塊消失、得分（基礎分 + 可選連對加分）、下一塊從頂端落下。 |
| **答錯** | 方塊消失、不計分、下一塊落下。 |
| **落地未答** | 方塊到底部仍未送出答案 → 該題不計分、方塊消失、下一塊落下；遊戲繼續（不 Game Over）。 |
| **結束方式** | 第一版：可採「玩到玩家按結束」或「固定題數（如 20 題）後結算」。 |

---

## 2. 視覺與操作

| 項目 | 說明 |
|------|------|
| **方塊** | 一個矩形區塊，中央大字顯示題目（如 12+17=?），可加邊框/陰影區隔。 |
| **下落** | 方塊自畫面上方以固定或漸增速度向下移動（每 frame 更新 y）。 |
| **落地線** | 畫一條「地面」線，方塊底邊 ≥ 該線即視為落地（未答則不計分、換下一塊）。 |
| **輸入** | 一個數字輸入框 + 送出按鈕（或 Enter 送出）；送出後清空輸入、立即判定並換題。 |
| **可選** | 方塊旁或底部「剩餘時間條」：隨下落縮短，強化時間壓力（第一版可省略）。 |

---

## 3. 題目與難度

| 項目 | 說明 |
|------|------|
| **題目來源** | 使用現有 `generateQuestion(options)`（`src/generator/index.ts`）。 |
| **選項** | `operation: 'add' \| 'mixed'`、`rangeMin/rangeMax`、`difficulty: 'easy' \| 'normal' \| 'hard'`。 |
| **題目字串** | 由 `question.a`、`question.b`、`question.op` 組成，例如：add→`+`、sub→`−`、mul→`×`、div→`÷`，結尾 `=?`。 |
| **難度漸進（可選）** | 每答對 N 題加快下落速度；或每 M 題提高 `rangeMax`。第一版可先固定速度與範圍。 |

---

## 4. 計分

| 項目 | 說明 |
|------|------|
| **基礎分** | 答對一題 +1 分（可依難度調整）。 |
| **連對加分** | 連續答對 3 題多 +1、5 題多 +2 等（第一版可簡化為「連對 3 +1」）。 |
| **落地未答 / 答錯** | 不計分、不計入連對。 |

---

## 5. 技術要點

| 項目 | 說明 |
|------|------|
| **路由** | 新頁面：`app/block/page.tsx`（或 `app/fall/page.tsx`）。 |
| **遊戲迴圈** | 使用 `requestAnimationFrame` 或 `setInterval` 更新方塊的 `y` 位置。 |
| **狀態** | 當前方塊對應一題 `Question`、方塊 `y`、下落速度、分數、連對數、是否暫停/結束。 |
| **碰撞** | 僅判斷「方塊底邊 ≥ 地面 y」即為落地；不實作方塊堆疊。 |
| **輸入** | `onSubmit` 時比對 `parseInt(輸入, 10) === question.answer`，並更新分數/連對、產生下一題。 |
| **下一題** | 答對 / 答錯 / 落地時：呼叫 `generateQuestion(options)` 取得新題，方塊重置到頂端、y 歸零。 |

---

## 6. 實作步驟清單

1. **新增頁面** `app/block/page.tsx`  
   - 標題、返回首頁、分數顯示、連對顯示（可選）。

2. **題目顯示**  
   - 抽一題 `generateQuestion({ operation: 'add', rangeMax: 20, difficulty: 'easy' })`（第一版固定）。  
   - 實作 `formatQuestionText(question)` → `"12+17=?"` 形式。

3. **方塊與地面**  
   - 一個 div 代表方塊，style 用 `transform: translateY(blockY px)` 或 `top: blockY px`。  
   - 地面線：一條橫線或「落地區」底邊 y 常數（如 `GROUND_Y`）。

4. **遊戲迴圈**  
   - `useEffect` + `requestAnimationFrame`（或 60fps setInterval）：每 frame 將 `blockY += fallSpeed`；若 `blockY >= GROUND_Y` 且尚未送出則視為「落地未答」→ 換下一題、重置 blockY。

5. **輸入與送出**  
   - 輸入框 + 送出（Enter 或按鈕）。  
   - 送出時若正確：加分、更新連對、換下一題、blockY 重置。  
   - 送出時若錯誤：不計分、連對歸零、換下一題、blockY 重置。

6. **計分與連對**  
   - 答對：score += 1；若 streak >= 3 則 score += 1（或自訂）。  
   - streak：答對 +1，答錯或落地 歸零。

7. **首頁入口（可選）**  
   - 在 `app/page.tsx` 加一個「落下方塊」或「方塊算術」的連結連到 `/block`。

## 8. 後續擴充（不做入第一版）

- 多欄落下方塊、方塊堆疊。
- 落地未答累積 N 次即 Game Over。
- 難度選項（速度、範圍、運算類型）放在畫面設定。
- 與 skill 權重 / 弱項整合（由 adaptive 抽題）。
- 紀錄成績到 persistence（如每日最佳分數）。
