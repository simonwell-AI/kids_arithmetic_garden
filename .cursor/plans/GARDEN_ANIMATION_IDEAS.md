# 花園動作動畫：華麗有趣改版建議

目前已有：澆水（水壺傾倒＋水滴＋水花＋閃爍）、施肥（搖擺＋粒子＋光環）、剪雜草（剪刀＋雜草淡出）、鬆土（叉子搖擺＋土粒＋閃爍）、噴霧（噴霧瓶＋霧狀＋水珠）、噴殺蟲劑（搖擺＋雲霧＋滴粒）、營養土（袋傾倒＋土粒＋閃爍）。以下建議可讓動畫更華麗、更有趣，且多數可用純 CSS 或少量 JS 達成。

---

## 一、通用強化（套用在多數動作）

| 建議 | 作法 | 效果 |
|------|------|------|
| **結束小驚喜** | 動畫尾聲加 0.3～0.5 秒的「完成閃光」：全畫面或植物周圍一圈 `box-shadow` / 偽元素從 scale(0.8) 擴到 1.2、opacity 1→0 | 每次動作結束都有「叮」一下的滿足感 |
| **輕微震動/彈跳** | 植物或花盆在動作觸發時加 `animation: small-bounce 0.4s ease-out`（translateY 0→-4px→0） | 像被碰到一樣，更有互動感 |
| **粒子數量與層次** | 現有粒子（水滴、土粒、霧滴）加倍或加一層「大＋慢」的粒子，錯開 delay；部分用 `blur()` 或半透明做景深 | 畫面更飽滿、不單薄 |
| **音效對齊** | 確保每個動作的關鍵幀（如水壺倒水、剪刀合起）與音效時間對齊，必要時用 `--duration` 與音效長度同步 | 華麗感一半來自「視聽同步」 |

---

## 二、分項動作建議

### 澆水 (water)
- **水花更活潑**：水花環不只變大，加一點 `rotate` 或 2～3 個不同大小的環交錯，或短暫 `scale(1.1)` 再縮回。
- **水滴軌跡**：水滴落下時加一層「拖尾」偽元素（細長條 opacity 漸變），或第二段 keyframes 讓水滴略帶橢圓形變形模擬速度感。
- **葉面反光**：在植物區加 1～2 個小橢圓 `gradient`，做 0.5s 的 opacity 0→0.6→0，模擬水珠反光。

### 施肥 (fertilize)
- **高級肥料更閃**：高級肥料除紫色光環外，可加「星星形」或小十字閃爍（多個 `::after` 不同角度、延遲），或 2～3 層光環由內而外擴散。
- **顆粒落地彈跳**：粒子落地時多一段 keyframe：到 80% 時在底部，85% 時略彈起（translateY -3px），100% 靜止，像小顆粒彈一下。
- **袋口光效**：肥料瓶/袋口處加一小段從亮到暗的 gradient 動畫，像有東西倒出來。

### 剪雜草 (weed)
- **草屑飛散**：剪除時除雜草淡出外，加 2～4 個小綠色碎片往不同方向 `translate` + `rotate` + `opacity → 0`，像草屑飛出。
- **剪刀閃光**：剪刀合起瞬間加一條白色/銀色線條（line 或 border）閃過，或剪刀刃上短暫 `filter: brightness(1.5)`。
- **震動反饋**：剪的瞬間整塊雜草區或花盆輕微 `translateX` 左一下右一下（5px～8px），配合音效。

### 鬆土 (fork)
- **更多土粒**：土粒數量增加，且分兩批：一批快、一批慢，落點略隨機（用不同 left/top 與 delay）。
- **小石頭/深色粒**：混入少數深褐色、略大的圓點，用不同 delay 落下，像翻出小石頭。
- **叉子反光**：叉子搖擺時加一層線性 gradient 從左掃到右的 `animation`，模擬金屬反光（0.3s 即可）。

### 噴霧 (mist)
- **霧氣層次**：兩層霧：一層大、慢、低對比（blur 大）；一層小、快、略明顯，交錯出現。
- **葉片掛珠**：在植物中段加 3～5 個小圓點，在噴霧中後段從 opacity 0 變為 0.8，像葉片掛上水珠。
- **噴霧瓶光點**：瓶身或噴頭處加一個小高光點，隨 swing 輕微移動。

### 噴殺蟲劑 (spray)
- **雲團更濃**：現有雲霧 keyframes 可加一階「擴散後略收縮」或多一層半透明圓，讓霧團更有體積感。
- **驅蟲/蜜蜂飛走**：若有蜜蜂或蟲，可在噴霧開始時加一段 0.5s 動畫：蟲/蜜蜂 `translate` 往畫面外飛出 + `opacity → 0`，像被趕跑。
- **藥劑色**：滴粒或雲霧可帶一點淡綠/淡藍色（例如 `rgba(180,220,180,0.6)`），與「水」區分。

### 營養土 (soil)
- **土粒落地堆疊感**：土粒不只落下，最後 20% 關鍵幀加輕微「彈跳」或「散開」（scale 略大再定住），像真的撒在土上。
- **袋口開合**：若有袋狀圖，可加一段袋口從閉到開再略回的 keyframes，與傾倒同步。
- **地面光澤**：土落下後，花盆土面加一層很淡的棕色高光，0.5s 內 opacity 0→0.3→0，像新土反光。

---

## 三、收成／開花專屬

- **收成**：已有慶祝粒子；可再加「植物微微縮小＋淡出」或「果實/花脫離」的 0.5s 動畫，再接現有彩帶與音效。
- **開花瞬間**：若可偵測到開花時機，可加 0.5s：花苞位置一個小光團放大＋淡出，然後開花圖出現，配合短音效。

---

## 四、效能與取捨

- **粒子數量**：同一畫面動態元素建議控制在約 20～30 個以內，多餘用「層次」（大而少、小而多）表現。
- **will-change**：只加在正在動的圖層（例如 `animating === "water"` 時才給 `.garden-animate-water` 加 `will-change: transform`），動畫結束後移除，減少吃 GPU 記憶體。
- **prefers-reduced-motion**：在 `@media (prefers-reduced-motion: reduce)` 裡將動畫改為 `duration: 0.01s` 或關閉次要粒子，保留主要反饋即可。

---

## 五、實作優先順序建議

1. **高 CP**：結束閃光、植物/花盆輕微彈跳、剪雜草草屑飛散、噴霧/澆水多一層粒子。
2. **中**：施肥顆粒彈跳、鬆土土粒加倍＋小石頭、殺蟲劑驅蟲飛走、營養土土粒彈跳。
3. **進階**：金屬/瓶身反光、葉面水珠/反光、開花瞬間光團。

可先選 1～2 個動作（例如澆水＋剪雜草）做為「華麗版試點」，再依反應擴到其他動作。
