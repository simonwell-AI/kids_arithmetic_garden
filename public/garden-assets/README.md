# 花園素材說明

此資料夾放置花園/商店會用到的圖片素材（種子、成長階段、花朵、工具、水壺、背包等）。
專案以「固定路徑 + 檔名」引用圖片，因此建議保持檔名一致。

## 建議資料夾結構

- `public/garden-assets/`
  - `coins/`：代幣圖示
  - `gargen_tools/`：工具與肥料（保持現有拼字）
  - `watering_can/`：水壺
  - `Backpack/`：背包
  - `pink_flower/`：粉色花朵成長階段
  - `sun_flower/`：向日葵成長階段
  - `grass/`：草地背景/裝飾

## 免費素材來源（可商用/教育使用為主）

| 類型 | 來源 | 備註 |
| --- | --- | --- |
| 種子/成長階段 | Vecteezy | 搜尋 "seed growing", "flower growth stages" |
| 生命週期 | Pngtree | 搜尋 "plant life cycle", "sunflower germination" |
| 花朵/種子圖示 | Freepik | 注意授權與標註需求 |
| 圖示 (PNG/SVG) | Noun Project | 搜尋 "flower seeds", "plant seed" |
| 通用圖片 | Pixabay, Unsplash | 多為免費可商用 |
| 遊戲素材 | Kenney, OpenGameArt | CC0 友善，適合遊戲 |

## 使用建議

- 若要替換素材，請保持檔名與路徑一致，避免程式端修改。
- 若暫時沒有素材，頁面仍可正常運作，但可能出現空白或 404。
